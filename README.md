## The Fine Print Worth Knowing

The server specs are shared across all plans: 2 vCPU and 2GB RAM per cPanel account, with IO capped at 50MB/s. That's completely fine for blogs, business sites, portfolios, WooCommerce stores with moderate traffic, and landing pages. Where you'd start to feel the limits is if you're running something like a high-traffic marketplace or a resource-heavy application that needs sustained I/O throughput. For those cases, VPS or dedicated hosting would be more appropriate anyway.

Free migration is included for up to 3 websites. Beyond that, there's an additional fee — contact their team for a quote.

Bonus you might not know about: customers can get a free `.xyz`, `.top`, or `.shop` domain for the first year by opening a support ticket after purchase.
